# 🎮 FocusGrid

An attractive, 3D claymorphic anti-distraction workspace designed for modern students. FocusGrid combines a silent, anxiety-free web frontend with a cross-platform mobile companion app to help you lock into deep flow states alongside peers.

---

## ✨ Core Philosophy & Features

FocusGrid changes the way we look at productivity. Instead of micro-managing your time with high-stress countdown timers, it emphasizes clean spatial structure and calm ambient indicators.

*   **⚡ Frictionless Flow State:** Enter focus zones silently without anxiety-inducing ticking countdowns. 
*   **🎨 3D Claymorphic Design:** Injected with custom soft Neumorphic HTML/CSS styling blocks to create a beautiful floating card interface that jumps off the screen.
*   **👥 Social Presence Accountability:** A dedicated peer grid layout that keeps you tethered to active study nodes in real-time.
*   **📈 Integrated Analytics:** Custom Plotly metrics tracking peak campus study volume patterns seamlessly.

---

## 📦 Tech Stack Architecture

The workspace is engineered across a dual-platform model built strictly on Python core engines combined with responsive web primitives:

*   **Web Console Application:** Powered by **Streamlit** + Injected **HTML5/CSS3** styling structures.
*   **Mobile Companion Frame:** Structured via **Flet** mapping responsive layouts into Flutter engines.
*   **Data & Visualization:** Managed through **Pandas** matrices and modern **Plotly** graphing wrappers.
